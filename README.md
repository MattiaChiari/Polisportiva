# 🏅 SportClub Manager (Gestionale Polisportiva)

![JavaFX](https://img.shields.io/badge/JavaFX-FF0000?style=for-the-badge&logo=java&logoColor=white)
![JSON](https://img.shields.io/badge/JSON-000000?style=for-the-badge&logo=json&logoColor=white)

Applicazione desktop sviluppata in **JavaFX** per l'amministrazione e l'organizzazione di una polisportiva. Il software fornisce un'interfaccia intuitiva per gestire l'anagrafica degli atleti, i palinsesti delle attività sportive, le iscrizioni ai corsi e l'analisi statistica delle affluenze.

## 🚀 Funzionalità Principali

### 👥 Gestione Atleti
* **Registrazione:** Inserimento di nuovi atleti con dati anagrafici, contatti e preferenze sportive.
* **Manutenzione:** Modifica e aggiornamento delle informazioni esistenti.
* **Rimozione:** Cancellazione sicura di un atleta dal sistema.
* **Visualizzazione:** Lista completa degli iscritti con dettagli di riepilogo.

### ⚽ Gestione Attività Sportive
* **Creazione Corsi:** Aggiunta di nuove discipline (es. Calcio, Pallavolo, Basket) specificando orari, giorni e descrizione.
* **Gestione Palinsesto:** Modifica ed eliminazione delle attività sportive.
* **Catalogo:** Elenco completo di tutti i corsi attivi.

### 📝 Iscrizioni e Assegnazioni
* **Iscrizione ai Corsi:** Associazione degli atleti a una o più attività sportive.
* **Storico Atleta:** Visualizzazione del profilo di un atleta con il dettaglio di tutti i corsi a cui è regolarmente iscritto.

### 📊 Rapporti e Statistiche
* **Analisi Affluenza:** Monitoraggio del numero di atleti iscritti per ogni singola attività sportiva, utile per valutare i corsi più popolari.

## 🛠️ Tecnologie Utilizzate
* **Linguaggio:** Java
* **Interfaccia Grafica:** JavaFX (Scene Builder / file `.fxml`)
* **Gestione Dati:** Serializzazione tramite file **JSON** per il salvataggio in locale.
* **Architettura:** Pattern MVC (Model-View-Controller) per separare la logica dalla UI.
